Backpack TF Tooltips

This is a neat little script that pulls TF2 item data from backpack.tf's and SteamPowered's APIs via PHP cronjobs, and turns parses item names into tooltips. The only thing you have to do is wrap any item name, or part of an item name in some span tags, like so:
<span class="tfPrice">Refined Metal</span> This was designed with forums that use BBCode in mind, but can easily be placed in any website that uses PHP & MySQL.
